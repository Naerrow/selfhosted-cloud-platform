## 원칙

- **1차 출처 인용 의무**: Proxmox API, AWS API, 한국 법규의 구체 사실은 공식 문서·법령 URL + 확인일 명기.
- **시의성**: 한국 규제는 Phase 종료 시점에 재확인한다 (R-005 완화책, ROADMAP Phase 0 Exit 게이트).
- **변경 절차**: 본문 정정은 [ADR-0010](../02-adr/0010-research-doc-correction-exception.md)의 4단계 분류 체계. 1차 출처 기반 정정 중 결정 영향 없는 것은 본문 직접 수정 + 머리말 개정 표기.
- **법적 면책**: 한국 규제 인용은 일반 정보 제공이며 법률 자문이 아니다 ([`../00-overview/vision.md`](../00-overview/vision.md) §7).

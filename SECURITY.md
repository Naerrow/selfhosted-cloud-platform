# Security Policy

## Reporting a Vulnerability

본 프로젝트는 Phase 0 (코드 작성 전 설계 단계) 진행 중입니다.
현 시점에서 보안 신고는 다음 두 경로로 받습니다.

- **취약점 신고 (비공개)**: GitHub Security Advisory의 "Report a vulnerability" 비공개 제출.
- **설계 결함·문서 사실 오류**: 일반 GitHub Issue로 신고. 처리는 [ADR-0010 분류 체계](docs/02-adr/0010-research-doc-correction-exception.md)에 따릅니다.

본 프로젝트는 1인 운영이라 응답 시간은 변동이 있을 수 있습니다.
사업화 단계 진입 시 별도 SLA를 README에서 안내합니다.
